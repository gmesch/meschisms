Slides in HTML

Write a slide set as an HTML document.
  • Each slide is one <div> element.
  • State is encoded in URL.
  • Build steps.
  • IFrames are loaded only once a slide is shown.
  • MathJax support for Math.
  • Interactive drawings on the slides.
  • Follow the presenter, including drawings, using #follow= URL parameter.

Slides in HTML

How to use.
  • Copy this readme.html as template to your git repo.
  • Map the slides.git repo as a submodule at ./slides.
